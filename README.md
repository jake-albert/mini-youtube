# mini-youtube

The project de-emphasizes things like detailed UI or modular, decoupled code in favor of a quick run-through of various Google-based cloud services:
* Cloud Run
* Cloud Pub/Sub
* Cloud Storage
* Artifact Registry
* Firebase Functions
* Firestore
